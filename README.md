
# 🚀 FUTURE_FS_02 — LeadFlow Mini CRM
### Client Lead Management System
**Future Interns Full Stack Web Development Internship — Task 02**

---

## 📌 Project Overview

LeadFlow is a simple yet functional **Client Lead Management System (Mini CRM)**
built to manage client leads generated from website contact forms.

It allows a business owner or admin to:
- View leads coming from website contact forms
- Update lead status (New → Contacted → Converted)
- Add follow-up notes for each lead
- Manage leads from a secure admin panel

---

## ✅ Features

| Feature | Description |
|---------|-------------|
| 🔐 Secure Admin Login | Username and password protected dashboard |
| 📋 Lead Listing | View all leads with name, email, phone, source and status |
| 🔄 Status Updates | Update lead status — New, Contacted, Converted |
| 📝 Notes & Follow-ups | Add follow-up notes for each lead |
| 🔍 Search Leads | Search leads by name, email or source |
| 🎯 Filter by Status | Filter leads by New, Contacted or Converted |
| 📊 Stats Dashboard | Live count of Total, New, Contacted and Converted leads |
| ➕ Add Lead | Add new leads from admin panel |
| ✏️ Edit Lead | Update existing lead details |
| 🗑️ Delete Lead | Remove leads from the system |
